# RecoverWell - Post-Surgery Recovery Tracker

A guided recovery companion app for post-surgery and physiotherapy patients.

## Features

- **Daily Exercise Tracking** - Check off exercises as you complete them
- **Pain Logging** - Track daily pain levels (0-10 scale)
- **Progress Photos** - Visual recovery tracking
- **Milestones** - Set and track recovery goals
- **Shareable Reports** - Generate reports for your therapist
- **Progress Dashboard** - View your recovery statistics

## Problem It Solves

Recovery from surgery is a day-by-day process with fluctuations. Existing apps fail to recognize this reality. RecoverWell provides:

1. Simple daily tracking without cognitive overload
2. Visual progress documentation
3. Professional reports for healthcare providers
4. Motivation through streaks and milestones

## B2B2C Model

- **Physiotherapists** can prescribe the app to patients
- **Clinics** can monitor patient progress remotely
- **Patients** get a guided recovery experience

## Tech Stack

- HTML5
- CSS3 (Modern, mobile-first design)
- Vanilla JavaScript
- LocalStorage for data persistence

## How to Use

1. Open `index.html` in a web browser
2. Log your daily pain level
3. Complete and check off exercises
4. Take progress photos
5. Generate reports for your therapist
